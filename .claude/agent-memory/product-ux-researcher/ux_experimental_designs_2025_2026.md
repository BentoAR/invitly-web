## Tendencias de Diseño 2025-2026

### 1. Kinetic Typography
- **Qué es**: Texto que se estira, tuerce, derrite en respuesta a scroll, cursor, voz
- **Implementación**: GSAP, Framer Motion, native CSS animations + variable fonts
- **Filosofía 2026**: "Purposeful motion" (movimiento con propósito, no caos)
- **Aplicaciones**: Letters scale/rotate con scroll, headlines responden a cursor, glitch-inspired transitions

### 2. Scroll-Driven Animations CSS Nativas
- **Tecnología**: CSS scroll() y view() timelines (scroll-progress + view-progress)
- **Browser support 2026**: Chromium + Safari 26 (Firefox aún no)
- **Performance**: Corren en compositor = buttery motion, GPU acceleration, off main thread
- **Ventaja vs JS**: No throttles ni rAF loops

### 3. Bento Grids Evolucionados
- **Evolución 2026**: Exaggerated corner rounding, micro-interactions dentro de cada tile
- **Advanced**: Subtle background tints por categoría (azul analytics, verde collaborative features)
- **Origen**: Apple-inspired → formato must-have para high-end web design

### 4. Glassmorphism Funcional
- **Transformación**: De flashy UI experiment → herramienta de jerarquía visual
- **Nueva dirección 2026**: Dynamic background blurs que shiftan con scroll
- **Paired con**: Materiality + depth sin sacrificar legibilidad

### 5. 3D & WebGL Cards
- **Técnicas populares**:
  - Parallax tilt effect (rotación 3D basada en cursor)
  - Floating cards con shadows + smooth return
  - Interactive 3D cards con GLB models + Three.js
  - Soft animated glow + reflection overlay
- **Implementación**: `rotateX`, `perspective`, layered divs, scroll-tied animations
- **Efecto**: Apple-like feel, premium UX

### 6. Liquid Metal & Blob Animations
- **Estética**: Irregular drops, dark liquid metal con textured surfaces
- **Movimiento**: Blobs cruzan pantallas, conectan entre sí, shapeshifting con interacción
- **Uso**: Portfolio websites, creative agencies
- **Inspiración**: Sci-fi traditions, cinematic quality

### 7. Infinite Horizontal Scroll
- **Codrops approach**: Seamless horizontal loop de cards posicionadas absolutamente
- **Advanced**: 3D infinite carousel con reactive background gradients (colores extraídos de paletas de imágenes)
- **Animation**: Cards "enter stage right" → hold → "exit stage left"
- **Upgrade path**: DOM/CSS/JS → GPU-powered WebGL con Three.js + shaders

### 8. Experimental Typography
- **Tendencia**: Bold experimental typography + vibrant color palettes
- **Técnica**: Saturated gradients, strong contrasts, expressive fonts
- **Aplicación**: Energía visual + jerarquía establecida
